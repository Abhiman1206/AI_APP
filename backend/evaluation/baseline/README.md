# Baseline Benchmark Artifacts

This folder stores immutable baseline benchmark outputs for Phase 0.

Required artifacts:
- baseline_summary.json
- baseline_per_doc.csv
- baseline_failures.json

Reproducibility checklist:
1. Record commit hash and branch.
2. Record Python version and package freeze.
3. Record benchmark command and input dataset manifests.
4. Record contract version from ../criteria_contract.yaml.

Do not overwrite baseline artifacts. Create a dated subfolder if re-running baseline.
